# 熬汤日记·呼吸·觉察 - 使用说明

感谢你选择「熬汤日记」！这是一款为阿斯汤加练习者设计的极简打卡工具。

## 📱 关于下载

**为什么是0.1元？**
- 小红书平台要求商品必须有价格
- 设置0.1元是为了合规上架
- 这不是收费，只是为了满足平台要求

**拍下后你会收到什么？**
- 小红书官方会自动发送网页app链接
- 无须注册，全平台可用（iOS / Android / 鸿蒙）
- 无需下载安装，直接在浏览器使用

## 💻 关于app

**为什么还没上架应用商店？**
- 这是我个人开发的独立项目
- 应用商店需要营业执照等企业资质
- 个人开发者暂时无法上架

**网页app也很好用！**
- ✅ 全平台兼容：iOS、Android、鸿蒙都能用
- ✅ 数据安全：使用浏览器本地存储，数据不会上传
- ✅ 持续更新：个人开发，快速迭代，根据用户反馈优化
- ✅ 离线可用：支持添加到主屏幕，像原生app一样使用

## 📲 安装到主屏幕（推荐）

### iOS 用户
1. 用 Safari 浏览器打开链接
2. 点击底部分享按钮⎋↑
3. 选择"添加到主屏幕"
4. 完成后可以像普通app一样使用

### Android 用户
1. 用 Chrome 浏览器打开链接
2. 点击右上角菜单⋮
3. 选择"添加到主屏幕"或"安装应用"
4. 完成后可以像普通app一样使用

## 常见问题

**Q: 数据会泄露吗？**
A: 不会。所有数据都保存在本地浏览器，同步功能支持手动导出/导入功能。

**Q: 需要付费吗？**
A: 目前完全免费使用。未来可能会推出付费会员，但现核心功能会一直免费。

**Q: 可以记录其他类型的练习吗？**
A: 可以！除了阿斯汤加，你也可以记录其他瑜伽练习、健身训练等。
