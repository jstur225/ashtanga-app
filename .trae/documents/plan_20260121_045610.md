# Ashtanga App UI & Logic Refactor Plan

## 1. Tab 1 (Practice) Header Refactor
- Remove `SyncButton` component and its logic from the main header.
- Center the logo icon and "熬汤日记·觉察呼吸" text using `flex-col items-center justify-center`.

## 2. UI Polish: Button Shadows
- Refine the practice option button shadows: change from heavy `shadow-[0_8px_32px_rgba(45,90,39,0.4)]` to a smoother `shadow-xl shadow-primary/15`.
- Ensure buttons have proper container padding to prevent shadow clipping.

## 3. Tab 2 (Journal) Date Logic
- Replace all hardcoded dates (e.g., `2026-01-18`) with dynamic `new Date()`.
- Update `MonthlyHeatmap` styling: past dates set to `text-foreground` (black), future dates set to `text-muted-foreground/40` (grey).

## 4. Timeline Font Scaling
- Increase the font size of practice notes in the `JournalTab` timeline from `text-xs` to `text-sm`.
- Adjust vertical spacing between timeline items for a more compact yet readable layout.

## 5. Tab 3 (Stats) Functional Fixes
- Re-link statistical calculations (`currentMonthStats`, `totalStats`) to the actual `practiceHistory` from `usePracticeData`.
- Fix the heatmap scrolling issue by adding `pb-32` to the `StatsTab` scroll container.

## 6. Data Management: "Data Capsule" Logic
- Swap Export/Import icons to match standard conventions.
- **Export Logic**: 
    - Copy data JSON to clipboard.
    - Show Toast: "已复制！这是您的‘数据胶囊’，请妥善保管。"
- **Import Logic**: 
    - Replace file upload with a text input/modal for pasting JSON strings.
    - Add verification and data restoration logic.
