## 📖 项目简介

**阿斯汤加打卡app**是一个极简的练习记录工具，专注于阿斯汤加瑜伽的打卡和身体觉察记录。

### 核心理念
**简单** - 专注打卡功能做到极致，不做加法

### 产品定位
- ❌ 不是瑜伽学习app
- ❌ 不是体式教学app
- ✅ 就是打卡 + 身体觉察的记录工具

### 目标用户
阿斯汤加练习者（练了很久的人，不是新手）

- ✅ 打卡计时功能（精准计时，支持暂停）

## 🛠️ 技术方案

### 平台选择（推荐）
**推荐：先做 Android** ⭐

**理由**：
- ✅ **成本极低**：$35-45（Google Play: $25一次性，域名: $10-20/年）
- ✅ **竞品空白**：Ashtanga Yoga Days只支持iOS，Android市场空白
- ✅ **不需要Mac**：Windows电脑即可开发
- ✅ **审核相对宽松**：Google Play比App Store容易通过

vs iOS：
- ❌ 成本高：¥5700-8700第一年（Mac: ¥5000-8000 + Apple Developer: $99/年）
- ❌ 竞争强：Ashtanga Yoga Days已占据iOS市场
- ❌ 需要Mac电脑

**营销角度**：
> "市面上唯一的阿斯汤加打卡app只支持iOS，我们终于出了Android版！"

### 技术栈推荐

| 阶段 | 方案 | 成本 | 时间 | 美观度 | 推荐度 |
|------|------|------|------|--------|--------|
| **原型** | v0.dev | 0 | 3-5天 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **MVP** | Claude Code + React | 0 | 1-2周 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 备选 | Flutter | 0 | 2-3周 | ⭐⭐⭐ | ⭐⭐ |
| 备选 | Streamlit | 0 | 1周 | ⭐⭐⭐ | ⭐⭐⭐ |

**推荐方案**：
- **阶段1（原型）**：v0.dev - 快速生成高颜值UI，3-5天
- **阶段2（MVP）**：Claude Code + React - 灵活开发，1-2周
- **后端**：Supabase（免费额度足够）
- **部署**：Vercel（免费hobby tier）

**不推荐**：
- ❌ Flutter - 对非技术用户太复杂
- ❌ Native开发 - 需要深入编程知识

详见：[app开发流程指南_非技术人员AI开发.md](./app开发流程指南_非技术人员AI开发.md)
